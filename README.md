# 🚀 Setup de Ambiente de Desenvolvimento (Ubuntu/Debian)

Este repositório contém um script Bash automatizado para configurar rapidamente um ambiente de desenvolvimento completo em distribuições Linux baseadas em Debian (como Ubuntu, Linux Mint, Pop!_OS, etc.).

O objetivo é poupar tempo após uma formatação ou na configuração de uma nova máquina de trabalho, instalando todas as ferramentas essenciais, IDEs, e utilitários de comunicação em uma única execução.

## 📦 O que este script instala?

O script mescla o uso do `apt` (com repositórios oficiais), `snap` (para garantir versões atualizadas de IDEs e apps desktop) e scripts de instalação direta.

### 💻 IDEs e Editores de Código

- Android Studio
